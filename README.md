# Web Scraping - AgendaWeb

Essa é por enquanto, uma ideia de uma atividade prática da disciplina de Linguagem de Programação II.

O objetivo do código é realizar web scraping no site AgendaWeb, extraindo links de worksheets (folhas de exercícios) disponíveis na página.

A ideia do projeto surgiu no meu dia a dia. Como trabalho dando aulas de Inglês, eu utilizo o site AgendaWeb para buscar atividades e lições de casa para os alunos. Com isso, estou tentando automatizar a coleta desses links, facilitando o acesso e organização dos exercícios.

O programa acessa a página, identifica os links disponíveis, exibe os resultados no terminal e salva os dados em um arquivo de texto.
